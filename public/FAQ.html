Question 2: Which language should I learn first or next?

Which language is easiest to learn?

Which is easiest to learn html, python, or java?

What is the hardest thing to learn in coding?

a) No one language is easier to learn coding with than another.

b) Learning the syntax of a language is a VERY MINOR part of learning to code.

c) Learning to think logically, to break large problems down into small bits that are then easy to code, learning to use functions and parameters properly and learning to write efficient code that is easy to understand - they are hard to learn and take time and LOTS of practice. You WILL NOT learn this any faster with Python than you will with C and C++

So what language should you learn first? Which ever one that you personally find the most aesthetically pleasing, the one that you have the most interest in learning and the one you are most likely to stick at when the learning gets TOUGH! HTML can't be learned in isolation from CSS, Javascript and PHP. All are required to create interactive and responsive web pages. PHP and Javascript have the same issues as detailed above.

Question 11: What is the best programming language to learn so I can get a job?

There is no one programming language that will guarantee you of a job. Different companies create different types of software, and different types of software require different programming languages.

And quite often, products use multiple types of software to operate them. Consider GridConnect smart home products. They use Android and iOS apps running on a mobile device to turn them on and off and change their settings.

But the devices themselves require special firmware running on them in order to make them react to signals from the mobile device and the state of internal electronics components.

So if you want the best chance of getting a job in the IT sector, then the more languages that you can competently program in, the better.

Question 13: Can I get a job with a boot camp certificate? What quals do I need to get a job?

If you want to get a job in larger corporations then more than likely such employers will require a Bachelors degree from a reputable university.

Smaller niche companies on the cutting edge of tech may well accept a portfolio of personal projects that demonstrate your programming capabilities well.

Few reputable employers in the west are likely to accept certificates issued by two bit boot camp companies located in India and Pakistan etc. These boot camp courses do nothing more than teach the syntax of a language. But simply knowing the syntax of a language does not make you a job ready programmer. And any reputable employers of programmers know this!

Question 14: What is the difference between a coder and a programmer?

NO - there is no difference between a coder and a programmer.

And any one who tells you is trying to manufacture difference where there is none. They are simply attempting to create status hierarchies, and claim the higher status for themselves.

It is like saying that a gardener is different to a landscaper. Either may create an entire gardens form scratch. Either may maintain existing gardens. They are the same job with different arbitrary labels.
